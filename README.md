# DACS2

DACS2 là một dự án nhằm phát triển một hệ thống quản lý dữ liệu và điểm danh nhập dang khuôn mặt. Mục tiêu của DACS2 là hỗ trợ người dùng quản lý dữ liệu một cách dễ dàng, an toàn và đáng tin cậy.

## Yêu cầu hệ thống

- Python 3.8 trở lên
- MySQL Server 8.0 trở lên
- Các thư viện Python cần thiết (xem phần Cài đặt)  
- Hệ điều hành: Windows 10 trở lên
- Phần cứng: Máy tính với cấu hình tối thiểu 8GB RAM, 2GHz CPU trở lên
- Webcam hỗ trợ nhận dạng khuôn mặt
- Trình điều khiển đồ họa tương thích (Yêu cầu có card đồ hoạ rời để xử lý hình ảnh hiệu quả)
